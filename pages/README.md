# Table of contents

---
- `/` : 게시물 리스트 + 바로가기 + 사이트 통계
- `/about` : 이력사항 말고 내 소개와 사이트 소개
- `/about/resume` : 이력서 + 이력사항
- `logs` : 매일 개발한 내용을 간단하게 트위터 처럼 토막글로 남기자
- `article` : 글 쓰기 연습
- `episode` : 나한테 벌어진 재미있는 일들

---
- 각 내용은 기본적으로 nuxt.js 의 라우팅 규칙으로 생성하며, Post Type 은 Wordpress 의 카테고리를 사용해서 분기처리.
